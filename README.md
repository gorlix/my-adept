# Ploopy Adept Configuration

Custom configuration of the QMK firmware for the Ploopy Adept trackball.

## Features

| Action | Functionality | Description |
|--------|--------------|-------------|
| Left click | Standard select | Primary mouse click |
| Right click | Context menu | Opens context menu with right button |
| Middle click | Middle click | Useful for opening links in new tabs |
| Orb rotation | Vertical scroll | Smooth scrolling of pages |
| Side button 1 | Back | Navigate to previous page |
| Side button 2 | Forward | Navigate to next page |
| Key + rotation | Desktop switch | Rotate the orb while holding key to switch desktops/workspaces |

## Installation

1. Clone this repository  
2. Compile the firmware using QMK  
3. Flash it onto your Ploopy Adept

## Requirements

- QMK Firmware  
- Ploopy Adept trackball

## Notes

Configuration tested on Arch Linux with KDE Plasma.
